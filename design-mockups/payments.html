Paid · Partial · Outstanding are DERIVED, not stored. There is no payment status column in the database — each label is recomputed from the two sums above every time the sheet loads (Paid = collected ≥ expected · Partial = 0 < collected < expected · Outstanding = collected is 0). They are UI-side labels: no payment-status value is stored or returned.

Paid Partial Outstanding → squared badge with a left rule — deliberately not the pill shape used by Pending Complete Cancelled which is a real enum (LessonStatus).
